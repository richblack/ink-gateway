# Obsidian Ink Plugin

An Obsidian plugin that integrates with the Ink-Gateway system to provide AI chat, semantic search, and intelligent content management capabilities.

## Quick Start

1. Install the plugin from Obsidian Community Plugins (coming soon) or manually
2. Configure your Ink-Gateway server URL and API key in settings
3. Start using AI chat, semantic search, and automatic content sync
